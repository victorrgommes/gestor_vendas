✅ PASSO 1: Criar repositório no GitHub (ONLINE)
================================================

1. Acesse https://github.com e faça login
2. Clique no "+" (canto superior direito) → New repository
3. Preencha os dados:
   - Repository name: gestor-vendas
   - Description: Aplicativo desktop para gerenciar vendas e exportar para Excel
   - Public (para que qualquer um possa ver)
   - ⚠️ NÃO marque opções de README/gitignore (já temos localmente)
4. Clique em "Create repository"

📋 GitHub mostrará comandos parecidos com:
   git remote add origin https://github.com/seu-usuario/gestor-vendas.git
   git branch -m main master
   git push -u origin master

🔑 TOKENS E AUTENTICAÇÃO (IMPORTANTE)
======================================

Se não tiver um GitHub Token:

1. GitHub → Settings (canto superior direito)
2. Developer settings → Personal access tokens → Tokens (classic)
3. Generate new token
4. Marque: repo, gist, read:user
5. Copie o token gerado
6. USE O TOKEN como senha no terminal
